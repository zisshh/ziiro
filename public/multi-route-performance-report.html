What Improved

  • Split every public page into lazy route chunks instead of one large shared route bundle.
  • Lazy-loaded and conditionally mounted the orbital service visualization so hidden desktop widgets do not run on mobile.
  • Deferred the Supabase client until contact/audit form submission.
  • Removed unused React Query and global toaster shell work from every route.
  • Hardened the Focus navigation after lazy-loading and paused orbital JS rotation in hidden tabs.

Rejected Experiments

  • Font trimming: average score fell from 91.0 to 90.7, so it was reverted.
  • Static legal pages: average score fell from 91.0 to 90.6, so it was reverted.
